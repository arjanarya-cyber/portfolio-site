# Arjan Arya — Portfolio

A minimal, modern portfolio site built with HTML, CSS, and vanilla JavaScript.

## Quick Start

```bash
# Open directly in browser
open index.html

# Or serve with Python
python3 -m http.server 8000
```

## Structure

```
portfolio-site/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
├── .gitignore
└── README.md
```

## Customization

- Update your name, bio, and projects in `index.html`
- Change colors in `css/style.css` under `:root`
- Add real project links and descriptions
